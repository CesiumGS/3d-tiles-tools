# Test data for `mergeJson3tz`

This directory contains tilesets for the specs of the `TilesetMerger3tz`
class that is used for the `mergeJson3tz` command.

The `tilesetA/B/C/D` directories contain the tilesets in their
unpacked form. The corresponding 3TZ files contain the respective 
tilesets as 3TZ.
